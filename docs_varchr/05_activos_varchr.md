# Activos de Información

## Activos Críticos del Hotel Costa Brava

El Hotel Costa Brava administra información esencial para su operación diaria. La pérdida, modificación o divulgación de estos datos tendría un impacto significativo tanto económico como reputacional.

| Activo | Importancia | Riesgo Principal |
|--------|-------------|------------------|
| Base de datos de clientes | Muy Alta | Robo de información |
| Reservas de habitaciones | Muy Alta | Alteración o eliminación |
| Datos personales de pasajeros | Muy Alta | Fuga de datos |
| Tarjetas bancarias | Crítica | Fraude financiero |
| Portal web | Alta | Interrupción del servicio |
| Servidor de aplicaciones | Crítica | Toma de control |
| Información de empleados | Alta | Robo de identidad |
| Respaldos del sistema | Alta | Pérdida de continuidad operacional |

## Riesgos para la Industria Hotelera

Las empresas del rubro hotelero son objetivos frecuentes debido a la gran cantidad de información personal y financiera que administran.

Entre los principales riesgos destacan:

- Robo de datos personales.
- Fraude con tarjetas bancarias.
- Secuestro de información mediante ransomware.
- Ataques al portal de reservas.
- Suplantación de identidad.
- Pérdida de disponibilidad del sitio web.

## Consecuencias para el Negocio

Un incidente de seguridad podría generar:

- Pérdidas económicas.
- Demandas legales.
- Sanciones por incumplimiento normativo.
- Daño reputacional.
- Cancelación de reservas.
- Pérdida de confianza de los clientes.
